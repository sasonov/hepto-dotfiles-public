# Learnings Index

Quick reference for finding relevant learnings by topic.

## Categories

- **devops** — Infrastructure, deployment, networking, Docker, systemd, security
- **coding** — Code patterns, debugging, architecture, testing, refactoring
- **research** — Research pipeline, web extraction, citation tracking, summarization
- **tools** — Tool-specific tips (terminal, browser, memory, cron, delegate_task)
- **general** — Cross-cutting patterns, context management, communication

## How to Use

Before starting a task in a category, read the relevant file:
- `devops.md` for infrastructure tasks
- `coding.md` for software development
- `research.md` for research and analysis
- `tools.md` for tool usage
- `general.md` for anything cross-cutting

Inject 3-5 most relevant tips as a user message near the start of the session.
